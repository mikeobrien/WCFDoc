Task Reference <WcfDoc> v1.0

<WcfDoc>

Wcf Doc is a tool that documents SOAP and RESTful WCF services.

Parameters

Attribute Type DescriptionRequired
Assemblies string Pipe seperated paths to the assemblies containg the WCF Service and Data Contract(s). True
Output string Path of the output file. True
Stylesheet string Path to the xml stylesheet. False
MergeFiles string Pipe seperated paths to xml files to be merged with the xml documentation prior to transformation. These can be explict or wildcard paths (IE: [Path]\\*.xml). False
WebsitePath string Path to the root of a service website. False
Config string Path to the service web/app.config containing service configuration. False
XmlComments string Pipe seperated paths to the xml comments files to include. These can be explict or wildcard paths (IE: [Path]\\*.xml). False

Examples

  • Document web services. 

        <Project xmlns="http://schemas.microsoft.com/developer/msbuild/2003">
        <UsingTask TaskName="WcfDoc.MSBuild.WcfDoc" 
                   AssemblyFile="[AssemblyPath]\WcfDoc.MSBuild.dll" />
        <Target Name="MyTarget">
            <WcfDoc
                Assemblies = "D:\Temp\Contracts.dll|D:\Temp\Services.dll|D:\Temp\Restful.dll"
                Output = "D:\Temp\Documentation.html"
                Stylesheet = "D:\Temp\Documentation.xsl"
                MergeFiles = "D:\Temp\MergeFiles\*.xml|D:\Temp\Summary.xml"
                WebsitePath = "D:\Temp\WebRoot"
                Config = "D:\Temp\WebRoot\Web.config"
                XmlComments = "D:\Temp\Restful.XML|D:\Temp\Contracts.XML|D:\Temp\Services.XML">    
        </Target>
    </Project>

Requirements
